# Bitcoin Trader Sentiment Analysis

## Objective
This project analyzes the relationship between Bitcoin market sentiment (Fear & Greed Index) and trader performance using Hyperliquid historical trading data.

## Datasets
1. Fear & Greed Index Dataset
2. Historical Trader Dataset

## Analysis Performed
- Data Cleaning
- Dataset Merging
- Feature Engineering
- Average Net PnL Analysis
- ROI Analysis
- Win Rate Analysis
- PnL Distribution Analysis
- Coin-wise Performance Analysis
- Statistical Hypothesis Testing

## Features Engineered
- Net_PnL
- Trade_Value
- ROI
- Profitability
- Hour
- Day

## Key Findings
- Highest profitability during Extreme Greed
- Better ROI during Extreme Greed
- Fear periods show stronger upside outliers
- Greed periods show higher downside volatility

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- SciPy
